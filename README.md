# Elective Brgy System

A comprehensive web application for managing elective subjects in barangay educational institutions. This system streamlines enrollment, subject selection, and administrative tasks for students and educators.

## 🎯 Features

- **Student Enrollment** - Easy enrollment management for elective subjects
- **Subject Management** - Add, edit, and manage available elective courses
- **Real-time Tracking** - Monitor enrollment status and class capacities
- **Report Generation** - Generate academic performance and enrollment reports
- **User Authentication** - Secure login for students and administrators
- **Responsive Design** - Works seamlessly on desktop and mobile devices

## 🛠️ Technologies

- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Styling**: CSS/Bootstrap
- **Language**: JavaScript/JSX

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/xdronnie/elective_brgy_system.git
   cd elective_brgy_system
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   Create a `.env` file in the root directory with necessary configurations

4. **Start the development server**
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`

## 🚀 Usage

1. Access the application through your web browser
2. Log in with your credentials (student or administrator)
3. For students: Browse and enroll in available elective subjects
4. For administrators: Manage subjects, view enrollments, and generate reports
