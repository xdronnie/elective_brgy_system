# Barangay Management System

This project is a barangay management system developed using **React Native** and **EmailJS**. It provides an efficient way to manage barangay-related tasks with the following key features:

- **Email Notifications**: Seamlessly send notifications via **EmailJS** to keep users informed.
- **User Authentication**: Secure user authentication is implemented using **Firebase**, ensuring that only authorized personnel can access sensitive information.
- **Cross-Platform Mobile Support**: Built with React Native, this application works smoothly on both Android and iOS devices, providing a versatile solution for barangay management.
